# Проект 20: Визуальное программирование агентов в n8n

**Студент:** Анар
**Стек:** n8n (Docker), Ollama (Llama 3), Open-Meteo API, Telegram API.

## Описание проекта
В рамках данного проекта разработан автономный погодный агент, который собирает данные о погоде в Алматы, анализирует их с помощью локальной LLM и отправляет структурированный отчет пользователю в Telegram.

### Основные возможности:
* **Интеграция с API:** Автоматическое получение текущих метеоданных (температура, ветер, осадки).
* **AI-аналитика:** Использование локальной модели Ollama для превращения JSON в читаемый прогноз.
* **Система уведомлений:** Доставка отчета в Telegram.
* **Обработка ошибок:** Реализован глобальный `Error Trigger`, который оповещает разработчика в Telegram, если в любой части цепочки произошел сбой.

## Архитектура Workflow
1. **Trigger:** Ручной запуск (Manual Execute).
2. **HTTP Request (Get City Name):** Получение точных координат Алматы (latitude/longitude) через API геокодирования.
3. **HTTP Request (Get Weather Data):** Использование полученных координат для запроса текущих метеоданных в Open-Meteo API.
4. **AI Agent:** Анализ данных через Ollama (Llama 3:1). Агент объединяет информацию из обоих запросов в связный текст.
5. **Telegram Send:** Доставка отчета пользователю.
6. **Error Handler:** Мониторинг ошибок через Error Trigger.

## Как запустить
1. Импортируйте файл `Anar_Project20_n8n.json` в свой экземпляр n8n.
2. Убедитесь, что Ollama запущена локально на порту 11434.
3. Настройте Telegram Credentials (API Token и Chat ID).
